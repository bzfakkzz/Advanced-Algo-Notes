$Mbps$，Mb per second，指每秒有x Mb

$1B=8b$



[0带宽 - 蓝桥云课 (lanqiao.cn)](https://www.lanqiao.cn/problems/1565/learning/)



小蓝家的网络带宽是 200 Mbps，

问使用小蓝家的网络理论上每秒钟最多可以从网上下载多少 MB 的内容。



分析：



$1B=8b$，网络带宽$200 Mbps$，每秒能下载$200Mb$，

每秒能下载$\dfrac {200} {8}MB=25MB$



```cpp
#include <bits/stdc++.h>

using namespace std;

int n;

int main()
{
    n = 200 / 8;

    cout << n << endl;

    return 0;
}
```

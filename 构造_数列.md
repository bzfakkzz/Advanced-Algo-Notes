[Problem - C - Codeforces](https://codeforces.com/contest/2031/problem/C)



分析：



如果n为偶数，$1^2=1$，每两个数字相同



否则当n为奇数，题目要求至少同一个颜色为2个，找最小的3个能满足平方和的数

得9 + 16 = 25，

最小的满足要求数组大小为27，

先填 `a[1]=a[10]=a[26]=1` ，再 `a[25]=a[27]=2` ，

再 `a[2~9]=a[11~18]=3~10` ，再 `a[19~23]=a[20~24]=11~13`

大于27，多的位置每两个填同一个数



```cpp
cin>>n;

if(n&1)
{
    if(n<27)puts("-1");
    else
    {
        a[1]=a[10]=a[26]=1; a[27]=a[25]=2;

        cnt=3;

        for(int i=2;i<=9;i++)a[i]=a[i+9]=cnt++;

        for(int i=19;i<=24;i+=2)a[i]=a[i+1]=cnt++;

        if(n>27)
        {
            for(int i=28;i<=n;i+=2)a[i]=a[i+1]=cnt++;
        }

        for(int i=1;i<=n;i++)cout<<a[i]<<' ';

        cout<<endl;
    }
}
else
{
    for(int i=1;i<=n/2;i++)cout<<i<<' '<<i<<' ';

    cout<<endl;
}
```

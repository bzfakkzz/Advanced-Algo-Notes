只能和相邻数操作，不能跨间隔操作

[Problem - C - Codeforces](https://codeforces.com/contest/2030/problem/C)



分析：



1. 第一个/最后一个是1，or 放在1后/前，整个式子为 1，A赢

2. 连续两个1，`1 1`
   
   > 两种情况：
   > 
   > 1. `or 1 or`
   > 
   > 2. `or 1 or 1`
   > 
   > `... = 1 or`，A赢

3. 不属于上面两种情况
   
   > `0 1 0 1 0 0 0`
   > 只要A将 or 放在1旁边，B用 0&1 变成0，B赢
   > `0 or 0 1 0 0 0`
